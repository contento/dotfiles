# dotfiles

Linux dot files

## References

* [wesdoyle/dotfiles](https://github.com/wesdoyle/dotfiles.git)

## Installing Dependencies

    pushd $HOME/dotfiles
    chmod +x install-base.sh && ./install-base.sh
    popd

## Install dot files

    pushd $HOME/dotfiles
    chmod +x .make.sh && ./.make.sh
    popd
